# Online Quiz Application

## Overview
This is a responsive Online Quiz Application developed using HTML, CSS, JavaScript, Bootstrap, and Node.js. It allows users to register, log in, select a course, take a quiz with multiple-choice questions, and view their final score.

## Features
- User Registration
- User Login
- Course Selection
- Quiz Instructions
- Timer-Based Quiz
- Multiple Choice Questions
- Automatic Score Calculation
- Responsive User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Node.js
- Git
- GitHub

## Project Structure
- index.html
- login.html
- register.html
- course.html
- instructions.html
- quiz.html
- result.html
- server.js

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Run the server using:
   ```
   node server.js
   ```
4. Open the application in your browser.
